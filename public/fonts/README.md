# GT Ultra Font Files

Place your GT Ultra font files in this directory.

## Required Files

- `GT-Ultra-Regular.woff2` (weight: 400)
- `GT-Ultra-Medium.woff2` (weight: 500)
- `GT-Ultra-Bold.woff2` (weight: 600)
- `GT-Ultra-Black.woff2` (weight: 700)

## Font Source

GT Ultra is a commercial font from Grilli Type. You'll need a license to use it.

If you have access via Adobe Fonts or another font service, update the `@import` statement in `src/styles/fonts.css` instead.

## Fallback

Currently using Inter as a fallback font, which has similar geometric characteristics to GT Ultra.
